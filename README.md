# 🛍️ Shopverse – React E-Commerce App

Shopverse is a modern e-commerce web application built using React.js.
It allows users to browse products, search items, and manage their shopping cart with a clean and responsive user interface 🚀

---

## 🌟 Features

* 🛒 Add to Cart functionality
* 🔍 Product Search
* 📦 Product Listing & Details
* ❤️ Wishlist / Favorites (if available)
* 📱 Responsive design for all devices

---

## ⚙️ Tech Stack

### Frontend

* React.js (Hooks)
* JavaScript
* CSS / Tailwind CSS

---

## 📂 Folder Structure

* `/src` → React components and logic
* `/public` → Static files
* `package.json` → Dependencies and scripts

---

## 🚀 How to Run

1. Install dependencies:

```
npm install
```

2. Start the application:

```
npm start
```

## 📊 Languages Used

* JavaScript
